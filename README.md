# Naveen Manikandan — Portfolio

Built with Vite, React, Tailwind CSS v4, and GSAP.

## Run it locally

```
npm install
npm run dev
```

Opens at http://localhost:5173

## Build for production

```
npm run build
```

Output goes to `dist/`.

## Deploy

Push this folder to a GitHub repo, then import it on vercel.com or netlify.com.
Both auto-detect Vite and deploy in a couple of minutes with a free subdomain.
Your resume is already in `public/resume.pdf` and your photo in `public/photo.jpg`.

## What to check before going live

- Update the LinkedIn URL in `Contact.jsx` and `Navbar.jsx` resume link if needed
- Swap `public/resume.pdf` whenever you update your resume, the filename stays the same
- The Earthquake Risk Analysis project links to your live Streamlit app, confirm it's still up
- Add real social links if you want Instagram/X in the footer
